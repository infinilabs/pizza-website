        Get index #

        Returns information about one or more indices

        Examples #

        Get the information of all indices:

        GET /_index

        Get the information of the index named my-index under collection my-collection:

        GET /my-collection/_index/my-index

        Request #

        GET /_index
GET /<target>/_index
GET /<target>/_index/<index>

        Path Parameters #

        • target
          (Required, String) Comma-separated, names of the collections to specify (wildcard supported)

        • index
          (Required, String) Comma-separated, names of the indices to get (wildcard supported)
